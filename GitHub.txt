Github is a service that allows users to host project files and folders. To access github one must visit github.com. Users will have to create an account to use its services. Through github you can create repository, a place where the project can be placed. You can set your github repository to be made public or private. Github is widely used by developers across the world. It's a very popular platform that's used to share or maintain software projects. Github will then allow you to access this through git. Git can be downloaded and installed. Git comes in many forms, command line, terminal, shell or  GUI.
